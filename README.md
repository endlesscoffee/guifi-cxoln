## Formats

The format used here is `Markdown`.

Use [pandoc](http://pandoc.org/) to easily convert the _CXOLN_ into many formats, such as: `.PDF`, `.ODT`, `.DOC`, etc.

For example, run `pandoc CXOLN.ca_ES.md -o CXOLN.ca_ES.odt`
